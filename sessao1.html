Curso de Desenvolvimento de Sistemas

O que é o curso?

O curso de Desenvolvimento de Sistemas forma profissionais capazes de projetar, desenvolver e manter sistemas computacionais, com foco em software, banco de dados e aplicações web e mobile.

Conteúdo do curso

  • Lógica de Programação e Algoritmos
  • Banco de Dados (modelagem e SQL)
  • Desenvolvimento Web (HTML, CSS, JavaScript)
  • Programação Orientada a Objetos
  • Desenvolvimento Mobile
  • Engenharia de Software e Metodologias Ágeis
  • Redes de Computadores e Sistemas Operacionais

Mercado de Trabalho

O profissional pode atuar como programador, analista de sistemas, desenvolvedor web/mobile, administrador de banco de dados, testador de software e muito mais.

Duração

O curso tem duração média de 2 a 3 anos, dependendo da instituição.
